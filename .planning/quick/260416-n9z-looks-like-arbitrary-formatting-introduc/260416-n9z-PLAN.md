**src/formatter.ts — Replace renderTemplate function (lines 30-71):**

Remove the entire `renderTemplate` function and replace with:

```typescript
const TEMPLATE_VARS = ['description', 'start', 'duration', 'tags', 'project'];

function renderTemplate(template: string, vars: Record<string, string>): string {
    return template.replace(/\$([a-zA-Z_][a-zA-Z0-9_]*)/g, (_match, name: string) => {
        if (TEMPLATE_VARS.includes(name)) {
            return vars[name] ?? '';
        }
        return `$${name}`;
    });
}
```

The `vars` record changes from `Record<string, string | null>` to `Record<string, string>` — empty optional fields now pass `''` directly (not `null`).
